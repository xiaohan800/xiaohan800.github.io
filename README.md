# Bytebeat 音频字符双用工具

在线访问：[Bytebeat编辑器](http://xiaohan800.github.io/index.html)

## 项目简介
这是一个集成了**Bytebeat音频生成器**和**Unicode字符显示器**的双功能在线工具。您可以通过编写数学和位运算表达式来实时生成独特的数字音频，同时也可以使用内置的Unicode工具探索和显示各种字符。

### Bytebeat 的历史
- **2011年9月18日**：芬兰程序员Viznut发明了Bytebeat，一种通过简短表达式生成音频的技术。
- **2011年10月5日到10月10日**：在此期间，许多爱好者编写了大量的Bytebeat表达式，形成了一个创作热潮。
- 虽然如今Bytebeat相对小众，但它作为一种独特的艺术和编程形式，其创造力与探索精神将永远留存。

## 核心控制按钮

页面底部包含四个核心功能按钮，用于控制音频播放。

### 1. 播放按钮
- **样式**：蓝色背景，带三角形播放图标，标注「播放」字样。
- **功能**：开始解析并播放当前Bytebeat表达式。`T`值将从**当前位置**继续递增。
- **状态提示**：底部状态栏会显示「正在播放...」。

### 2. 倒放按钮
- **样式**：紫色背景，带双向三角形图标，标注「倒放」字样。
- **功能**：音频以倒序方式播放，`T`值会反向递减，生成独特的反向音效。
- **注意**：此操作不影响表达式内容，仅改变时间变量`T`的走向。顶部状态区会显示「倒放」的播放方向。

### 3. 停止按钮
- **样式**：红色背景，带正方形图标，标注「停止」字样。
- **功能**：立即停止音频输出，进入静音状态。
- **关键特性**：停止时，`T`值会**定格在当前数值，不会归零**。
  - 例如：`T`运行到`2,979,544`时点击停止，`T`将保持该数值不变。
  - 再次点击「播放」时，音频会从该位置继续往后计算。

### 4. 重置按钮
- **样式**：绿色背景，带循环箭头图标，标注「重置」字样。
- **功能**：将`T`值强制归零，并停止音频输出。
- **注意**：编辑器的输入内容、示例表达式、Unicode显示器、播放速度等其他逻辑**不会被重置**，仅重置时间计数器`T`。
- **用途**：适合重新从头开始运行一个表达式。

## 播放速度调节

本编辑器提供**双模式速度调节**，满足从粗糙到精细的控制需求。

### 1. 指数速度滑动面板
这是一个独特的**指数级**速度调节面板，允许您以极高的精度调整播放速度。

- **面板显示**：
  - 当前速度（如 `1.2144037459x`）
  - 对应的指数公式（如 `10^0.084`）
- **调节方式**：
  - **滑动调节**：按住并上下拖动/触摸滑动面板，实时调整速度。灵敏度默认为 `0.001`，可精确控制指数值。
  - **快捷加减**：
    - 「减0.5」「减0.1」：快速降低速度。
    - 「加0.1」「加0.5」：快速提升速度。
- **指数范围**：`10⁻³⁰⁰` 至 `10³⁰⁰`，支持超精细调速。
- **特性**：调节后立即生效，音频播放不中断。

### 2. 播放速度控制区
- **快捷按钮**：
  - 绿色「加速」按钮：提升播放速度倍率（乘以2）。
  - 紫色「减速」按钮：降低播放速度倍率（除以2）。
  - 棕色「重置速度」按钮：恢复默认速度（1x）。
- **预设档位**：`0.125x / 0.25x / 0.5x / 1x / 2x / 4x / 8x / ???x`，一键切换常用速度。
- **状态同步**：顶部状态区、滑动面板、速度控制区均会实时显示当前速度。

## 状态与采样信息

顶部和中部状态栏提供了丰富的实时信息：

- **语法检查**：显示当前表达式的语法状态和**字节大小**（如「语法检查通过！108B」）。
- **播放状态**：显示「播放中」「暂停」「停止」等状态。
- **播放方向**：显示「正放」或「倒放」。
- **当前采样率**：显示实际播放的采样频率（如 `9715Hz`）。注意，此数值会根据速度倍率动态变化。
- **当前T值**：显示当前采样地址（如 `2,979,544t`），并会根据数值大小智能切换显示格式（如 `1.23e6t` 或 `2,979,544t`）。
- **播放时间**：显示已播放时长（如 `372.443 秒`）。您可以通过修改代码顶部的配置，将其切换为「XX年XX月XX天...」的完整时间格式。

## 表达式编写规则

Bytebeat的核心是使用变量`T`（代表时间）来生成音频样本。您可以在输入框中输入合法的数学与位运算表达式。

- **基础运算符**：`+` `-` `*` `/` `%`
- **位运算符**：`&` `|` `^` `<<` `>>`
- **函数支持**：表达式内部可以使用 `Math` 对象的函数，如 `Math.sin`，`Math.cos` 等。
- **多语句支持**：您可以使用分号`;`分隔多个语句，并以最后一个表达式的值作为输出。
- **立体声支持**：返回一个包含两个值的数组，如 `[ (t>>4)&255, (t>>6)&255 ]`，即可生成立体声音频。
- **步长控制**：新增的步长控制模块允许您指定`T`值每增加多少单位计算一次音频样本（最小为1）。调整步长后，音频会实时生效，无需重启播放。

### 示例表达式：
- `t & (t >> 8)`
- `(t * 5) & (t >> 6)`
- `(t >> 4) ^ (t << 3)`
- `t * 3 & 255 | t >> 5`
- `t*(((t>>12)|(t>>8))&(63&(t>>4)))` (经典Bytebeat)
- `[ (t>>4) & 255, (t>>6) & 255 ]` (立体声示例)

## 实时波形图与信号分析

工具内置了实时波形显示器，帮助您从视觉上理解音频信号。

- **时间域**：显示音频波形的幅度随时间的变化。
- **频率域**：通过FFT（快速傅里叶变换）展示音频的频谱分布。右侧会列出当前频谱的**TOP 5峰值频率**，底部面板会显示「总体系数」（总能量）和「分贝条」。
- **无信号模式**：当没有播放音频时，波形图区域会切换为**Unicode字符显示模式**，在左侧大田字格中动态循环显示Unicode字符，右侧则显示字符编码和当前时间。

## Unicode 字符显示器

这个工具还集成了一个功能丰富的Unicode字符显示器。

- **核心功能**：在输入框中输入或粘贴文本，下方区域会实时显示对应的Unicode字符。支持包括Emoji在内的所有Unicode字符。
- **多语言界面**：支持简体中文、正體中文、English、日本語等多种界面语言。
- **本地化设置**：可以为显示的字符设置语言属性，以优化不同语言字符的渲染。
- **字符样式**：提供多种字体样式控制，包括**粗体**、**斜体**、**小型大写**、**斜线零**、**数字形式**（等宽/比例/旧式）以及**样式集**（SS01-SS04）。这些样式不仅应用于字符显示区，也会应用到上方的T值显示区域。
- **字体支持**：内置了多种高质量字体，包括 `Times`, `Hancom`, `TH-P0`, `TH-P2`, `TH-MingJ`, `MiSans` 等，以覆盖不同语言的显示需求。

## 其他功能

- **自动保存**：您的表达式、速度、音量、字符设置等会自动保存到浏览器的localStorage中，刷新页面或重新打开后不会丢失。
- **数据清除**：点击「清除数据」按钮可以一键清空所有保存的设置，恢复到初始状态。
- **随机小贴士**：页面中间会随机显示一条与编程、社区或Bytebeat相关的有趣文字，点击它可以切换到另一条。
- **内存占用/排行榜**：底部区域会实时统计各个类别的表达式数量，并生成一个按表达式个数排序的「种子排行榜」。

- **循环**： 增加了循环起始和循环结束，使其可以限制 t 值最小和最大的范围。

## 注意事项

- **音频API**：本工具使用 `ScriptProcessorNode` 进行音频处理，该API在现代浏览器中已被标记为“已废弃”，但在大多数浏览器中仍能正常工作。它适用于创建实时音频生成器，但可能在性能上不如新的 `AudioWorklet`。
- **性能**：某些表达式在极高的速度倍数下可能会消耗较多CPU资源，导致界面卡顿。请根据您的设备性能适当调节。
